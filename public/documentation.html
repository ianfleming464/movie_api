Wax on, wax off

A 1980s Movie API

The API will provide users with access to information about different movies, directors and genres. Users will be able to sign up, update their personal information, and create a list of their favourite movies


Request URL HTTP Method Request body format Response
Return a list of ALL movies to the user localhost:8080/movies GET None JSON object holding data about all the movies
Return data (description, genre, director, year, image URL) about a single movie by title to the user localhost:8080/movies/[title] GET None JSON object holding data about a single movie
Return data about a genre (description) by name/title localhost:8080/movies/[genre] GET None JSON object holding genre data about a single movie
Return data about a director (name, birth year, death year) by name localhost:8080/movies/[director] GET None JSON object holding director data about a single movie
Allow new users to register localhost:8080/movies/users POST JSON object holding user data, for example: {name: 'Ian Fleming', password: ****, email: ian@fleming.com, DOB: 1981-07-07 } JSON object holding user data that was added, including an ID
Allow users to update their user info (username, password, email, DOB) localhost:8080/movies/users/[username]/[password]/[email]/[DOB] PUT None Text message inducating relevant info was updated
Allow users to add a movie to a list of favourites localhost:8080/movies/users/favourites POST JSON object holding movie data, for example: { title: 'Red Heat', genre: { action, comedy }, director: Walter Hill, image URL: ****, year: 1988 } JSON object holding movie data that was added, including ID
Allow users to remove a movie from their list of favourites localhost:8080/movies/users/favourites/[title] DELETE None Text message indicating that the relevant movie was deleted
Allow existing users to deregister localhost:8080/movies/users DELETE None Text message indicating that the relevant user was deleted
