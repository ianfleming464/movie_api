Wax on, wax off

A 1980s Movie API



The API will provide users with access to information about different movies, directors and genres. Users will be able to sign up, update their personal information, and create a list of their favourite movies


Request URL HTTP Method Request body format Response
Return a list of ALL movies to the user /movies GET None JSON object holding data about all the movies
Return data (description, genre, director, year, image URL) about a single movie by title to the user /movies/[Title] GET None JSON object holding data about a single movie
Return data about a genre (description) by name movies/genres/[Name] GET None JSON object holding genre data about a single movie
Return data about a director (Name, Born) movies/directors/[Name] GET None JSON object holding data about a director
Allow new users to register /users POST JSON object holding user data, for example: {Username: 'IanFleming', Password: '******', Email: 'ian@fleming.com', Birthday : '1981-07-07' } JSON object holding user data that was added, including an ID
Allow users to update their user info (Username, Password, Email, Birthday) /users/[Username]/[Password]/[Email]/[Birthday] PUT None Text message indicating relevant info was updated
Allow users to add a movie to a list of favourites by ID /users/[Username]/favourites POST JSON object holding movie data, for example: { title: 'Red Heat', genre: 'action', director: 'Walter Hill', image URL: '****'', year: '1988' } JSON object holding movie data that was added, including ID
Allow users to remove a movie from their list of favourites /users/[Username]/favourites/ DELETE None Text message indicating that the relevant movie was deleted
Allow existing users to deregister /users/[Username] DELETE None Text message indicating that the relevant user was deleted
